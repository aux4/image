#### Description

The `convert` command changes the format of an image. It supports common formats including JPEG, PNG, WebP, TIFF, GIF, and BMP. The target format is specified with the `--format` flag.

If `--output` is not provided, the output file is generated by replacing the input file's extension with the target format (e.g., `photo.jpg` becomes `photo.webp` when converting to WebP).

#### Usage

```bash
aux4 image convert <input> --format <format> [--output <file>]
```

--format  Target format: jpg, png, webp, tiff, gif, bmp
--output  Output file path (default: input name with new extension)

#### Example

```bash
aux4 image convert photo.jpg --format webp
```

```text
Converted photo.jpg to webp -> photo.webp
```

```bash
aux4 image convert photo.jpg --format png --output assets/photo.png
```

```text
Converted photo.jpg to png -> assets/photo.png
```
